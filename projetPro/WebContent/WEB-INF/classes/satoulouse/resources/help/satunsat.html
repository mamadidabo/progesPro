SAToulouse has solved the problem.

Either you have obtained :

  • "the set of formulas is satisfiable"
SAToulouse gives you a model/valuation for it.

If SAToulouse gives "p, q, r", it means that the valuation v defined by
v(p) = v(q) = v(r) = 1 and v(s) = 0 for all propositions s different from p, q and r,
is such that v satisfies all the formulas.
Intuitively, the valuation v is a solution for the constraints represented by the formulas.


In the case where there is a lot of propositional variables, you can only watch the truthness of some propositional variables. You can select a subset of propositional variables by typing a common portion of the namesof the propositional variables you want to watch.


  • "the set of formulas is unsatisfable"
There is no valuation v such that v satisfies all the formulas. Generally speaking, it means that the constraints represented by the formulas can not be satisfied alltogether.
